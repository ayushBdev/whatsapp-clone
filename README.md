### Introduction

It is made using React, Node.js, Express & MongoDB. The App is clone of WhatsApp and it is a simple real time messaging app that allows users to:-

-Login / Sign Up

-Chat with friends

-Search and Add users to chat 

-Search for chat from sidebar

-Real time updates
